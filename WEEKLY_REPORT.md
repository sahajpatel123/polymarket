# WEEKLY_REPORT

Passive visibility for long-run unattended operation. Overwritten weekly by the
autonomous loop. Not an action request.

## Week of (UTC)

_Not yet generated — first weekly report due after 7 days of loop uptime or on
the next Monday UTC cycle, whichever comes first._

## Template fields

- System uptime / last cycle timestamp
- Tier-1 items completed this week
- Tier-2 PRs opened / still pending (`PENDING_REVIEW.md`)
- Paper P&L and key risk metrics (`scripts/paper_metrics.py` stdout)
- Dependency / security audit flags
- Credentials / certificates approaching expiry (if any tracked)
